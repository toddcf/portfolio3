Responsive Font Sizes

Here's how I maintained the desired font size ratio for each heading. (Hint: I set the ratios once and only adjusted the root font size in each CSS media query, significantly reducing the amount of code I needed to write.)

1. Decide on Font Size Ratio

First, I decided on a ratio, using Type Scale. There was some trial and error involved here because there were many other variables to take into account. Font, font weight, and the length of the text content, which could vary from heading to heading and from page to page.

Suffice it to say I landed on Augmented Fourth, which is a ratio of 1.414.
